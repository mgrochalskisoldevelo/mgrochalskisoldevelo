## About Me

My journey in tech began with building mobile experiences in Xamarin and .NET, but my focus shifted toward high-impact global solutions at SolDevelo. Having contributed to foundational initiatives like GovStack, I now serve as a member of the OpenLMIS core team. I focus on developing and scaling the platform using Java and PostgreSQL, while managing Docker-based deployments and DevOps on AWS. By supporting multiple national-scale implementations, I ensure this open-source logistics software creates resilient health supply chains where they are needed most.
